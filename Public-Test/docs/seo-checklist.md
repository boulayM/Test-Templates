# SEO checklist (Public-Test)

## Technique
- [ ] title unique sur chaque page publique
- [ ] meta description presente
- [ ] canonical present et coherent
- [ ] robots `index,follow` (public) / `noindex,nofollow` (auth/account)

## Social
- [ ] Open Graph: `og:title`, `og:description`, `og:image`, `og:url`
- [ ] Twitter: `twitter:card`, `twitter:title`, `twitter:description`, `twitter:image`

## Donnees structurees
- [ ] JSON-LD present uniquement sur pages indexables pertinentes
- [ ] schema coherent (pas de donnees fictives trompeuses)

## Performance SEO
- [ ] verifier LCP/CLS/INP sur pages publiques critiques
- [ ] verifier poids JS et images (build + stats)

## Validation
- [ ] verification Lighthouse (Performance/SEO/Best Practices/Accessibility)
- [ ] verification preview social (Open Graph / Twitter)
- [ ] verification indexabilite (robots + canonical)
